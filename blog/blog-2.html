How to Select the right Machine Learning Algorithm

  • Computer
  • Machine Learning
  • Data Science
  • artificial intelligence

Machine learning is half art and half science. For machine learning, there is no one solution or one approach that fits all. Many factors can influence our choice to pick a machine learning algorithm.

Some problems are very specific and require a unique approach. E.g. a recommender system. While some other problems are very open and need a trial and error approach. classification and regression are very open. They could be used to build more general sorts of predictive models. Let us look at some of the factors that can help us narrow down the search for the machine learning algorithm.

Before we start looking at different algorithms, you need to have a clear and better picture of our data, the given problem, and the constraints.

Know the data:

  • Summarize and visualize the data
  • Means and Medians helps to check the central tendency
  • Correlations for relationships
  • Standard deviation helps to check the spread out
Data visualization:
  • Scatter plots help to understand the bivariate relationships
  • Box plots help to identify outliers
  • Density plots help to show the spread of the data
  • Heat maps help to check the missing values
Clean the data:
  • Deal with missing values. Check if the missing values are at random or not
  • At random: We can replace using the mean/median/zero value of the data based on the distribution
  • Not at random: We can replace using mean/median/extreme value and then create another column to flag those missing values
  • There are thousands of techniques we can use to address missing values. There is a separate chapter on feature engineering
  • How to handle outliers: Very common on multidimensional data: Outliers can be the result of bad data collection, or they can be legitimate extreme values
Prepare the data:
  • Use PCA to reduce the dimensionality
  • Rescale the variables
Categorize the problem:
  • Input
    • If the data is well-labeled, it is a supervised problem
    • If the data is not labeled, it is an unsupervised problem
  • Output
    • If the output is a number, it is a regression problem
    • If the output is a class, it is a classification problem
    • If the output is in groups, it is s clustering problem
Knowing the constraints:
  • Know your data storage capacity to store your model or gigabytes of data
  • Understand your learning rate of your model
Now we have a clear understanding of our data, we can identify the algorithms. Some of the factors affecting the choice of a model are:
  • Model scalability
  • Model accuracy
  • The model meets the business goals
  • The building and prediction time of the model
  • Understanding the complexity of the model
Some common machine learning algorithms and some of their applications:
  • Linear regression: Sales prediction, and Credit score analysis
  • Logistic regression: Churn prediction, and Fraud detection
  • Decision trees: Bank loan defaulter detection, and Churn prediction
  • K-Means: Clustering analysis
  • Principal Component Analysis (PCA): Dimensionality reduction
  • Support Vector Machines: Stock market prediction, and Text Categorization
  • Naive Bayes (My Fav.):Text Classification, Sentiment Analysis, Recommendation System , and Face Recognition
The Scikit-learn cheat sheet is very handy!
